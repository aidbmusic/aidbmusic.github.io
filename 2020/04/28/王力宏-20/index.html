　　<白纸>

　　忙碌的宣传行程并不会降低了力宏制作专辑的质量，紧接着第六张专辑『不可能错过你』推出，展现跨国合作成果，国际知名小提琴手叶加濑太郎跨刀演奏的「流泪手心」，是近年来王力宏一首令人荡气回肠的大型情歌，同时也得到第11届金曲奖的最佳作曲人提名。

　　与小室哲哉、张惠妹、莫文蔚 音乐人惺惺相惜的合作

　　另外一项国际合作，是在1999年底，受到日本天王制作人小室哲哉的赏识，在由小室哲哉制作的一张欧亚美三大洲同步推出的公益单曲「Happiness x 3 Loneliness x 3」，担任中文版的编曲及主唱。音乐合作的邀约越来越多，除了小室哲哉之外，在张惠妹的新歌+精选专辑中也收录王力宏的作品「爱，永远不会消失」。2002年时更进一步为莫文蔚『i』专辑中的「那可不一定」这首歌担任创作及制作。

　　<永远的第一天>

　　在亚洲城市不断飞行转移的忙碌行程下，王力宏整理出内心很深刻的种种感受，完成了第一张包办全部制作的专辑『永远的第一天』。专辑中另一首话题作品是王力宏重新改编重唱「龙的传人」，王力宏首次就自己美国出生中国人的背景为歌曲主题，而且王力宏将这首歌的编曲赋予摇滚节奏的生命力，在2000年让这首歌以新面貌出现。为了配合这首的节奏，在MTV拍摄时，王力宏首次挑战了从来没有尝试过的舞蹈，而且是强调爆发力感的舞步，MTV的成果也获得一致好评。

　　大制作电影的配乐制作

　　同年，王力宏的事业版图首次跨足电影，演出国际合作的动作片【雷霆战警】，除了演出之外，王力宏还首次制作并主唱电影主题曲「China White」，除了制作诉说自己心情的歌曲之外，首次向为特别项目制作主题曲挑战。

　　<唯一>

　　2001年的『唯一』，王力宏创新制作概念，以王力宏式的Live为主轴，大量在歌里披露出他的成长感想，扩大了在自我创作的格局，「唯一」唱得深入，又表现出历经磨练的王力宏唱腔。

　　最骄傲的进化─不思议演唱会

　　在进行这张专辑宣传同时，王力宏正准备着2002年最重要的一场演唱会—在红磡的首次售票大型演唱会【222不思议演唱会】，别出心裁的华丽表演及舞蹈，精采的曲目大受好评，也是王力宏对自己舞台表演功力、创意、体能..等方面的一大突破，经过这场演唱会的洗礼，王力宏从歌手、音乐人、演员又进化到全能艺人的位子。一路以来的进化，全部完整的收藏在这张精选辑内。

　　⊙最值得乐迷收藏的5大理由

　　◎26首最完整

　　横跨BMG、福茂、SONY三大厂牌，收集26首为完整王力宏历年代表作。

　　◎16首最精华

　　从「情敌贝多芬」、「好想你」、「在每一秒里都想见到你」、「公转自转」、「不可能错过你」、「流泪手心」、「永远的第一天」到「唯一」，1995年以来每张专辑，全部主打歌一次收录。

　　◎8首最典藏

　　「Like A Gun Shot」电影蜘蛛人亚洲版主题曲、「Happiness x 3 Loneliness x 3」与小室哲哉合作跨国单曲、与范晓萱合唱「雪人」、电影雷霆战警英文主题曲「China White」..等。王力宏分散在其它作品中的各种创作，历年特别版单曲全部物归原主，珍贵典藏。

　　⊙视觉也在进化

　　专辑─简约但充满表情语言

　　这次专辑的视觉，着重在表现王力宏的〝思想表情〞。采用最简单的棚内摄影，但是最难的抓取神态的手法─捕捉在歌坛七年的王力宏，经历过的成长的表情。从刚进入歌坛的阳光少年到今天充满想法的音乐人，岁月虽让王力宏的外观有所改变，但不是老态，相反的是一种推进的，洗炼的新的面貌。

　　另外专辑里呈现一些陪着王力宏一路进化的随身物品，比如他因为忙碌的工作而弄到像圣经一样厚的护照、看的出为了一个音符的情绪而经历过王力宏各种操练的小提琴、陪他写下每一首歌以及和歌迷交流的计算机..等。这些东西在不知不觉中纪录了这七年的进化，除了音乐之外让我们最直接感受王力宏进化的见证。

　　MV─动静皆宜的精采演出

　　为了要配合「WHY」这首开朗的节奏与画面丰富的歌词，MV里出现了七个场景，而王力宏也配合歌词故事，做了趣味十足的表演，这是在王力宏以前的MV里少见的逗趣表现，用肢体演出音乐的标情，可以从「WHY」的MV里看出王力宏的进化。

简介

　　用一首歌谣唤醒灵魂中的民族自豪感 王力宏《南京，南京》

　　一段朗朗上口的旋律，一阙简单又充满故事的歌词，王力宏将“南京”谱写成了一首史诗般的歌曲。

　　对于一个往来了20多年的城市，总会有一些回忆在心头，难以忘怀；就如同那些熟悉的美景、美食、以及美丽的人和事。在“龙的传人2060”来到南京之际，在奥地利美丽的多瑙河畔，诞生了这样一首歌《南京，南京》。

　　初听之下，就能感受到力宏为这首歌投入了很深的感情；他将对南京的印象、对南京的情怀、对南京的爱意都融入了歌曲当中。歌词里那句“对你的爱谱成一首曲”更是完美诠释了力宏本人对南京的心声告白。这是他来自灵魂深处的对历史的感知和对文化的讲述，无论谱曲还是填词，你都能体会到他对这座城市的爱意至深。

　　不仅如此，力宏把自己的创作原稿伴随着旋律发到了个人的社交网站上，很快歌曲就被广大的受众赋予了更新的定义。这是一首为南京崛起而创作的歌曲，于是伴随着南京的美景、南京的美食、南京的风土人情，南京的变化，赋予了南京新的生命力，体现着城市的发展变迁。

　　在“龙的传人2060”南京站结束后，力宏亲自找到了很多带有地方特色的艺术表演者，共同努力；在原有音乐的基础上，加上了更具有南京特色的地方说唱内容；使得音乐更加有生活气息，而且更接地气，让音乐的力量能够渗入到每个人的精神中去；也让音乐成为一种榜样的力量，感染每一个人心中的民族自豪感，就像网友的口号一样，这是一首让“南京崛起”的歌！

　　王力宏睽违四年的全新专辑《你的爱。》发行在即，在公布专辑的首波主打歌之前，特别让[裂心]一曲提早释出抢听，当做送给歌迷的新年礼物。许多歌迷看到[裂心]这个歌名曝光时，纷纷表示大感讶异，没想到力宏身在幸福的家庭生活中，却在此时推出歌名看起来如此忧郁的一曲。
