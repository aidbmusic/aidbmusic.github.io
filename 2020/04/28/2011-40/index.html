不允許哭泣的場合

信息

又名: No Crying

表演者: 魏如萱

流派: 流行

专辑类型: 专辑

介质: CD

发行时间: 2011-11-10

出版者: 亚神唱片

唱片数: 1

条形码: 4712523871048

其他版本: 不允许哭泣的场合 （全部）

豆瓣评分: 8.1

简介

　　预购赠品: 魏如萱2010年香港《优雅的刺猬》演唱会现场录音CD

　　2010年「优雅的刺猬」专辑赢得中华音乐人交流协会【十大优良专辑】与华语音乐传媒大奖之【评审团专业致敬奖】等肯定。2011年春天「在哪里？」的单曲，我们看见一次比一次更诚实的魏如萱，原来坐在门后面的女孩是敏感又忧愁的。接连在内地与香港和台湾的巡回演出，让众多乐迷们为之惊艳也更加肯定了魏如萱现场演唱的千面女伶魅力。

　　2011年秋天剧场配乐/编曲名家陈建骐要再次让我们听见更广角的魏如萱「不允许哭泣的场合」，新专辑的封面拍摄也特地找来波兰摄影师Lukasz Wierzbowski，给魏如萱增添了似华沙美人鱼那般勇敢却又奇异的温柔气味。视觉则由目前正在伦敦大学修习艺术创作也是充满神秘感的Waterfall瀑布杂志主编小8来统筹，或许魏如萱的Every Teardrop能像Coldplay般慑人心魄。

　　「不允许哭泣的场合」，魏如萱用「标本」来形容这张专辑。

　　“他们将动物的尸体制作成标本，栩栩如生彷彿没有离开一样”

　　“歌在完成的那一刻，也代表故事已经说完了。”

　　有些记忆想忘却忘不了，借着制作标本就能将记忆隔离，「不允许哭泣的场合」是魏如萱寄托某些记忆制作的标本。

　　首先，从内到外的记忆皮囊里极满的情感和骨架都必须重新逐一拆解和经过破坏的过程。

　　麻醉 /

　　魏如萱与键盘手张瀚中共同谱写的曲子，宛如好莱坞经典电影歌曲的〈溶话〉里，风度翩翩的绅士旋律加上她蹑手蹑脚的气音，大方率性又撒娇扭捏，连耳垂尽是管弦乐的泡沫，诱人要踮起脚尖跟着旋转飘舞。

　　川烫 /

　　吉它手韩立康继「优雅的刺猬」〈麋人〉的曲子让魏如萱发展出有角的声线，〈三个字〉里电吉它与铜管齐响，加上魏如萱歌词中暗讽着现今部份年轻人的人际关系，你一定马上就能感受叫人气得想要乱叫又跺脚的情绪。

　　剔骨 /

　　由香港The Marshmallow Kisses双人组合其一的Peter编曲，〈一时〉像棉花糖中间藏着夹心的惊喜口味，疲累瞬间退散，不由自主想拍手又使人似醉酒般脸颊泛红，词曲及演唱都像踩着菱形步，小心，一不小心你就会跳针。

　　剥皮 /

　　饥饿艺术家Oliver负责编曲的〈脱光光〉节奏如连绵起伏的云层和太阳般一会儿亮一会儿又暗了的捉摸不定。这原是魏如萱随性创作的短诗，隐喻着人性里「明知山有虎，偏向虎山行」的盲点也让人心有戚戚焉啊。

　　缝合 /

　　引人入胜的是键盘手张瀚中创作的演奏曲〈抽屉〉，探戈据说起源于情人之间的秘密舞蹈，这样顿错感强烈的断奏式演奏紧接制作人陈建骐谱曲和吉它手徐千秀编曲的〈陨石〉像双人魔术般猜不透的旋律走向，待袭而来是惊人歌德派的暗黑面貌，魏如萱挑战不同以往的曲风，她是只刺猬或称她为野兽都还是优雅的，在俐落的一瞬间〈陨石〉即掳获你的双耳。

　　去油 /

　　制作人陈建骐少见的Trip-pop曲风加上诗人夏宇吟读诗集「Salsa」其中一篇「排队付帐」，夏宇认为诗集《这只斑马》《那只斑马》中〈勾引〉这篇相当适合人来性感的魏如萱，这样迷幻又妩媚的强烈风格，在台湾乐坛也算是特立独行，一向被笑称我行我素的外星人魏如萱，自称不运动又迷糊状态的爵士声线，令人很难不上癮。

　　漂白 /

　　彷彿凑近耳畔呢喃的〈晚安晚安〉，则缓步连结着她的声线与法语间独特的亲密关系，温柔但强烈地在睡前告白。

　　风干 /

　　魏如萱首次与雷光夏合作的〈我们〉，当你听见大提琴的低鸣，请慢慢随着海浪的潮起潮落走向这两位爱做梦的她们的梦境里。

　　固定 /

　　《飞鸟与鱼》故事众所熟知是《世界上最遥远的距离》，魏如萱将眷与恋的惆怅写进〈飞鸟〉，编曲则由「在哪里？」词曲创作〈If〉来自香港的Jackie负责，扣人心弦的吉它如羽毛般的浪回荡犹如掀起记忆中那段不舍的痴恋，即使相爱也不会是恋人，当朋友也是奢求的宿命彷彿引出雷光夏所创作〈我们〉的前传。

　　组装 /

　　继〈我爸的笔〉魏如萱出乎意料的逻辑见解，使口语的形容词再度变成趣味的口头禅，吉它手韩立康为〈吼呦〉打造的编曲，一定会让你不经意在洗澡或走路时随口就哼起，当然也不难想像现场演出时台上与台下一起吶喊的画面。

　　浸泡 /

　　魏如萱首次尝试编曲，制作人陈建骐认为简单的钢琴就如同简单的寓言般深刻，〈呜〉叙述猎人与猎物之间情感交错，在短短文字中也感受到其怪诞思绪，虽是绵密轻声却又错综如刀刃的抽离感，放在专辑结尾〈呜〉犹如烈酒般柔中带劲。

　　收藏 /

　　“我想忘了我想忘了的，但我忘了我是忘不了的。”

　　“记忆的标本虽然是死亡的，但这段记忆同时也用另一种方式继续活在我心里的抽屉。”

　　我一边听一边有个奇异的感觉，我觉得我还在超级市场排队等付账呢，可娃娃根本不想等，她已经出发去勾引了 —— 诗人 夏宇

简介

　　生命中最重要的十個寶藏你發現了嗎？

　　華麗搖滾天團F.I.R.2011音樂鉅作

　　F.I.R.第六章【亞特蘭提斯】

　　嶄新華麗的搖滾曲風

　　極度震撼的視覺

　　F.I.R.飛兒樂團2011全新力作

　　第六章【亞特蘭提斯】

　　每個音樂人心中都有一個理想世界，F.I.R.亦不例外

　　成軍多年歷經許多起伏、面對許多外界的干擾、內心的掙扎、市場的惡化…唯一不變的是F.I.R.對音樂的熱情與堅定！他們相信唯有發自內心的音樂，才是感動自己也感動別人的！

　　整整一年的時間，F.I.R.在忙碌的行程中仍不斷的創作，從累積的上百首原創中挑出了最精彩的10首歌曲。這一次我們將再次發現F.I.R.音樂的美好、喚回最美麗的音樂理想國-「亞特蘭提斯」。

　　推薦曲目

　　首波強打【亞特蘭提斯】

依然

信息

表演者: 李健

流派: 民谣

专辑类型: 专辑

介质: CD

发行时间: 2011-12-10

出版者: 金牌大风

唱片数: 1

条形码: 9787798985641 ISRC(中国): CNF181165100

豆瓣评分: 8.3

简介

　　《李健:第5张创作专辑 依然》是“音乐诗人”李健即将在12月10日交出自己的全新作品，这也是他的第五张个人创作大碟。在过去的两年中，李健走上了更加宽广的舞台，却仍不改梦想的初衷，继续保持着生命的澄澈和真性情的本色，对音乐的赤诚依然不变，对生命的询问依然不止，对亲情的追寻依然不灭。生活未变，李健依然。

　　作为一个创作歌手，李健一直坚持，发片不是音乐的全部，热爱生活与热爱音乐同样重要，于是，这两年他的轨迹遍布世界各地。从香港、澳门到宝岛台湾，从南非的开普敦到瑞士的苏黎世，从俄罗斯贝加尔湖畔到德国的柏林墙，从美国西岸艺术中心旧金山到世界的浪漫之都巴黎，这些来自不同种族的文化深深打动着他，一路思考，一路珍藏，没有一定的起点和终点，这是生活最本质的模样，更是难能可贵的创作源泉。在旅行与工作交织的700多天时间里，李健不慌不忙如期兑现诺言，新专辑应时出炉指日可待！

　　新专辑首波主打曲《我始终在这里（But I Still Love You）》，依然由李健包办词曲，在风格上却完全不同以往的平静悠扬，字里行间充满了对生活的“狠”态度——不论经历怎样的不平与磨难，都用一颗依然守候的心去接纳，即使“这勇气快耗尽我的生命”，但“生活仍然要继续”。在现实生活，其实人的情感往往不由自己可以控制决定，经历越多就越难以割舍。对出道十年的李健而言，红与不红，他始终在认真歌唱！面对人生的多姿与无常，冷峻的现实总让人身感无奈，而李健正是用音乐传达着自己对这个世界的希望——温暖而真挚、坚守而不绝望，随岁月的变迁，他的音乐也让我们更加懂得每一个转折的幽微，这些都源自于他对生活的热爱和对生命的尊重。一切正如他在“音乐风云榜”颁奖典礼摘下最佳男歌手时，面对华语乐坛的低迷不前所言，“抱怨毫无意义，惟有逆流而上”!而李健用清亮的嗓音实践着世间的悲苦和诗意，“不可预知的人生，我始终在这里迎接，无悔命运的波折与坎坷，但我依然爱你。”

　　在这张新专辑中，李健用婉转、清澈的声音向我们传达着更加丰富的生活内涵。它让我们想起了那些丢失、深藏、遗忘的理想，我们依然不断坚持；也让我们想起了最美的、深爱的、纯真的爱情，我们依然没齿难忘，还有那些无数正在或将要完成的“人生必修课”，高高低低，我们依然在不断经历生与死的诀别，离别与相聚的重逢：我们每一个人行走在那些无法停止，并且始终存在的情感中，步履维艰却收获丰硕。

　　为了精益求精，李健不惜远赴万里前往英国伦敦，在Metropolis Studio这间诞生过BLUE、Bon Jovi、Sting、Michael Jackson、Madonna、Lady GaGa等国际巨星经典专辑的世界上最伟大的录音棚中，李健完成了整张专辑的后期制作。

　　另外，在这张新专辑中，李健首度携手国际吉他大师杨雪霏合作，流行音乐与古典吉他交相辉映，将最东方的浪漫与多情演绎的如梦似幻。还特别收录了之前被听友广泛关注的由陈凯歌执导大型歌舞剧《希夷之大理》主题曲《珍爱深深》，这首歌由日本配乐大师久石让谱曲，这也是专辑中唯一一首不是由李健作曲的作品。

　　改变、不变、都来自你熟悉的李健；可以追随一生的温暖与澄澈；让人义无反顾爱上的治愈系美声，11首诚挚之作。诗意、深邃、激昂、冷静！诗意的纪录方式；动人的人文关切；深情的醉心吟唱；故事仍然在继续；李健让我们依然带爱前行！

　　★珍藏3款《一直走，在路上》李健旅行卡，无论身处何处，都会《心升明月》。

　　★共赴2年之约白岩松、于丹、姚晨诚意聆听至情推荐，《我始终在这里》等你。

　　★1句《谢谢你》李健亲笔分享创作心得，敞开心扉娓娓道来。
