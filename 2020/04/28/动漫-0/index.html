One more time,One more chance 「秒速5センチメートル」Special Edition

信息

又名: 秒速5厘米主题曲 / Byosoku 5cm

表演者: 山崎まさよし

流派: 原声

专辑类型: Single

介质: Audio CD

发行时间: 2007-3-3

出版者: NAYUTAWAVE RECORDS

唱片数: 1

条形码: 4988005464798

相关电影: 秒速5厘米

豆瓣评分: 9.3

简介

　　十年多之后的今天，『One more time, One more chance』配合新海诚的新电影重新发售特别版。熟悉JPOP的朋友应该知道，这首曲子曾是山崎まさよし主演的电影「月とキャベツ」的主题曲。也是他的代表曲目之一。

　　凝结了新海监督想最描绘的东西的作品「秒速5厘米」。

　　把现在这里有的“日常”的风景做为舞台，尝试用动画片表现出每天谁都能感觉到现实感。

　　正因为如此，新海监督自己很强地认识到做为主题歌的乐曲的重要性。

　　新海监督所想象的主题歌——「谁都知道，对现代人们来说有普遍性的乐曲」。

　　从过去到现在的许多的流行的歌曲，找寻了适合这个条件的乐曲

　　最终走到了山崎まさよし的「One more time, One more chance」面前。

　　在新海作品那情感溢出的风景里，配以这伤感却美丽的旋律。影片结束时响起的歌声有种力量让人感动……

　　单曲中还收录了天门所编曲的一曲”雪の駅”，还有初次音源化的”弾き語りVer”。
