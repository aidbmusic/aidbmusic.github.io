Venture

简介

　　时间是为了定位现在而创造的伪概念，人类像乘坐着宇宙中最快的交通工具，以不可思议的速度打败数字，进行迭代。而让我们进化的，是打破原本平稳无聊状态的欲望。

　　千篇一律的风格、含糊不清的态度、语焉不详的说辞……鹿晗站在这些的反面，玩一个粉碎无趣的游戏。守着山洞里的财宝固然安全，却也让骑士变成恶龙。是冒险家穿过未知海域发现新大陆，才让人看见世界的轮廓。鹿晗在每一次的音乐大航海里，燃烧着从不止步的冒险精神，让他区分于其他单薄的人设，成为混沌中令人一眼定睛的鲜活角色。

　　《V》一如既往承担着开拓的责任，鹿晗经历种种而后成长，以锐利的音乐之刃切割迷雾，试图唱出更多人与人之间关系的结构，更深入寻找音乐的可能性。

　　《如果》精致的Contemporary R&B，时髦的情歌句型

　　延续《某时某刻》的幕后阵容，由Santell和Picard Brothers（Maxme Picard与Clement Picard）为《如果》担纲制作，以入耳即化的温柔旋律突出鹿晗的声音色彩，辅以更为犀利的电子音效，点破恋爱行至分崩离析时，「想触碰却收回手」的姿态。对你的舍不得，是在自己的对话框里任凭心情风起云涌，也克制地按下delete。这是我和你的关系里，我最后的秘密。

　　相比于在恋爱的语境里充满过剩的情绪，鹿晗更善于用更为轻盈的叙事来展现当下年轻人的行事风格。学会接受爱的到来与消失，是成长的必经之路。《如果》以精细的编曲和演唱方式，展现出梦幻而易碎的年轻人的恋爱形态和成长后的处理方式。

　　《敢》态度重磅出击，有话直说的trap招式

　　是不是经常会遇见不懂分寸、没有界限的人来打扰你的节奏？或者听见那些躲在暗处的偷窥癖咀嚼着你生活的细枝末，大放厥词？是不敢直视小丑的眼睛，对杂音习以为常，还是用真实的态度为自己想要的世界投票？

　　全新歌曲《敢》用最时髦的trap曲风，通过犀利的rap，松弛而充满趣味的编曲，淋漓尽致地展现了鹿晗的耿直率性，哪怕在亿万人注视的镁光灯聚焦下，鹿晗也选择“有话直说”，用音乐坦诚表达，在现象里挖掘更多深意，为所有感同身受而唱。

简介

　　关于《I》－我

　　在经历（Xperience）、探索（Xplore）、冒险（Venture）、想象（Imagination）后，我们终于与“自我”再次相遇。不再盲目追寻千篇一律的剧情设定，无需瞻前顾后执着所谓正确的选择，关于成长的版图，无限延伸，由“我”定义。

　　作为鹿晗第二张专辑“XXVII”（27）系列的收官之作，最新数字MINI专辑《I》，从“我”出发，归于27岁的青春。这张专辑不仅是鹿式音乐的一个小结，更是送给每一位90后的礼物——无论是18岁，还是27岁，有过经历与探索，敢于冒险与想象，我们成为了现在独一无二的自己。

　　此次《I》专辑中的两首歌曲，在曲风上经历了前面4张专辑的风格积累后，回归到Bass music结构，但同时又将Trap和Dance-pop风格并置，与美式电子风格糅合，突出自我成长，引导音乐审美，拓展音乐视野，也再次丰富了“鹿式音乐”的架构。

　　《触发（Set it off）》

　　极为纯正的美式风格和性感唱腔，典型的Future bass风格，跳跃而充满磁性的唱法与背景绵长的合成器音色形成巧妙的对比，男性荷尔蒙爆表。而每一段Flow恰到好处的强弱顿音处理，更是彰显出鹿晗独有的音乐气质。制作人Rumours 和Phil 作为当前电音界的领军人物，此次与鹿晗的合作更是为我们带来最正统最前沿的电子音乐。

　　充满张力的音色中隐藏着收敛又坚定的进攻性，荷尔蒙爆表的性感唱腔中却也稳稳地存留一丝细腻与温柔。像是耳语，亦像午夜寂寥时不经意的撩动。年轻的自我，势均力敌的较量，我们屏住呼吸，因为一切一触即发。

　　《零界点（On fire）》

　　迷幻冷酷的EDM Trap曲风，搭配意气昂扬的歌词，充满趣味的编曲，每一个字都如鼓点般敲在心上。《零界点 （On fire）》此次邀请华语新兴音乐人AR合唱Rap部分，共同玩趣音乐，带来新的惊喜。“因为I will go higher，天空没有任何界限”。

　　作为90后代表的鹿晗，正通过不断探索不同的音乐风格，尝鲜各种组合碰撞创作的可能性，为华语乐坛带来了新的音乐元素，也表达了当下年轻音乐人独立的态度和审美。年轻的我们，各自精彩，态度鲜明，眼中燃烧着梦想的烈焰，不断突破自我到达未知的境界。This is my city，this is my pride！

简介

　　《Imagination》取义“想象”，预示着鹿晗在音乐探索路上，打开限制，不再停留于自我、过往的理念进化。创作力从方向明确的线性发展，迈向更广阔空间维度的自由运动。这既是鹿晗跨入音乐新领域的暗示，也是创作力空间打开的Key Word。就让鹿晗带领我们打开想象，一起打破固有现状，一起遇见未来。

　　《Imagination》的两首歌曲，歌曲立意不在有具体所指，更多的是从内心和想象出发的延伸。音乐风格上，两首作品，一首温柔忧郁，一首流畅欢快，两种风格，两种情绪，都是鹿晗在音乐世界中的无限想象。

　　想象-第I章：假如《时差（On call）》不只是距离

　　R&B是音乐世界的荷尔蒙，再加上爵士的助力，更使得音乐本身散发出极大的魅力，《时差》就是这样一首作品。首度尝试的优雅的电声爵士风格，配搭上时髦的Contemporary R&B。既是对一贯深受认可的“鹿式情歌”的延续，又是一次充满格调的新尝试。但鹿晗从不止步于安全地带，开场极明显的Looping电声效果在华语乐坛中着实是极为大胆的处理手法。贯穿全作的Looping效果构建起的电声氛围背景，为歌曲营造出一层微凉的忧郁情绪。而响指效果的R&B节奏处理则抢耳耐听，与鹿晗清冽的嗓音完美融合。

　　时差，地理学概念，却也是彼此的情感隔阂。我们与我们所爱的父母、恋人、朋友似乎总是存在时差。即使近在咫尺，也都各自为了生活奔波，宛若隔着无尽的光年。《时差》精准捕捉住现代人的情感缝隙，描绘出你我的不安。干净的配器、氛围化的音乐处理和鹿晗细腻的演绎，将种种复杂精简成一股单纯的想念呈现在我们眼前。仿佛问句的结尾略显突然，却留下意犹未尽的的韵味，让人忍不住一再按下Replay键。

　　想象-第II章：玩酷《夜行记（Say it）》

　　面对选择时，我们会大声Say it，勇敢做自己；还是在心里默默告诉自己“Save it，Nobody care you”？成为国王亦或是流浪的艺术家，都是自己的选择。可无论选择哪一种身份，每个人都是自己世界中的国王。

　　《夜行记》是与《时差》截然不同的一首作品，带着零负担的清爽公路Feel。春天就该有俏皮的气息，这是派对的玩酷开场曲。犀利的独白，充满趣味的编曲，淋漓尽致地展现了鹿晗的耿直率性，带着张扬，带着最自然流露出的欢愉。肆意挥霍的汗水和青春，就在此刻，大声Say it。

　　有两首旗开得胜的精彩单曲当前导，接下来还有一连串不可思议的惊喜等着乐迷来体验！揭起序幕的「Eraser」，直接秀出不输阿姆的白人饶舌技艺，又能伴随演唱的好歌喉，酷劲十足地在一开场即下一道猛药；嘶声力竭演绎「Dive」，沙哑高亢的音频伴随蓝调吉他弹奏，对于诚心接纳的真情挚爱大声呼唤，令人无比动容；浪漫到底的「Perfect」，飘扬出的柔顺弦乐如此醉心美丽，量身打造一支新世代的结婚进行曲；将传统居尔特民族之音融入嘻哈流行区块，飙出大胆又雀跃起舞之作「Galway Girl」；组织大型管弦乐团伴奏的「Happier」，情绪澎湃地撩起碎心恋曲，透露对刚分手女友的不舍与依恋；回归民谣舒适度的「Hearts Don`t Break Around Here」，Acoustic吉他频率足以让一颗对爱不变的心持续保温取暖；简单钢琴辅佐的「Supermarket Flowers 」，闭上眼睛细细品味的同时，不知不觉眼眶已经湿润；红发艾德玩起Beatbox，一旁的萨克斯風和小号热闹陪衬，借着「Barcelona」来趟充满阳光的异国旅行吧！还有选用非洲打击乐器的「Bibia Be Ye Ye」、居尔特之音再度环绕的「Nancy Mulligan」，一再令人诧异红发艾德大步跨出多元丰富的新节奏，感受他那畅快激昂与新鲜活力并行的缤纷洒脱，沉沦在他那热切拥抱梦想理念与情爱诗篇的动人乐章中！

时差 On call various versions collection

信息

表演者: 鹿晗 / 严晴 / 国际首席爱乐乐团 / CUCA人声乐团 / sususu / Cattzim / 坡上村 / DACH / 劳国贤

流派: 流行

专辑类型: 专辑

介质: CD

发行时间: 2017-04-25

出版者: 风华秋实

唱片数: 1

豆瓣评分: 7.9

简介

　　在音乐的世界里，不同的音乐风格犹如时区。跨越音乐风格也如跨越时区一样，需要克服众多的不适，也需要更多的勇气和冒险精神。尽管原版的《时差（On Call）》是一首标志明显的“鹿式情歌”，可年轻自可以有更多可能，所以可塑性极高的鹿晗还会继续尝试更多种不同的曲风，绝不会甘心只停留在一种曲风之上。这张《时差 On call various versions collection 》合辑正是这样的尝试。

　　跨越音乐风格的“时区” ，7种全然不同版本的《时差（On Call）》从各种音乐类别中汲取其风格、乐器、以及韵律，令流行音乐、民族音乐甚至传统音乐相粹结融会。充满冒险精神，打破了固定的音乐风格常规。这张合集突破单纯的流行音乐的概念，跨界至古典、器乐、地下电子等众多不同音乐领域，不但拓展了鹿晗的音乐世界，也赋予了作品更多的可能性和想象空间。

　　这不仅是鹿晗生日季对歌迷的音乐礼物，更是对音乐的一次“心灵回馈”。

　　曲目：

　　1、时差（On call）Chinese style version

　　中国民乐改编，古筝、琵琶、南萧来演绎R&B，真真正正的“中国风”。

　　2、时差（On call）String quartet version

　　作品经过古典室内乐的加持，四把提琴用充满韧性的音色表现音乐中最细腻的心韵。

　　3、时差（On call）A cappella version

　　纯人声作品——所有的声音均来自人声。通过多声部丰富的和声细致演绎原作丰富的情愫。

　　4、时差（On call）sususu remix

　　与中国地下电子音乐的一次跨界，鹿晗的音乐视野并不只是局限于线上的音乐作品，更会外延至更前卫地下的领域，探寻流行音乐的孵化之地。

　　5、时差（On call）坡上村 cover

　　跨界国内独立音乐圈之作。作品风格独树一帜。乐队成员更包括鹿晗《Reloaded》专辑的录音师。

　　6、时差（On call）E cappella version

　　听起来像EDM，但其实所有的声音都是人声。人声实验之作，跨界电子音乐风格。

　　7、时差（On call）Guitar version

　　曾为鹿晗《致爱》担纲吉他演奏的著名吉他手劳国贤再度出山，献上吉他器乐版本的《时差》。因为吉他的加入，让氛围化的作品有了乐器演奏的温度与热情。

简介

　　★水星音乐奖得主暌违五年第三张专辑

　　★荒芜中执拗燃烧的浪漫

　　★都会喧嚣间最亲密的孤寂

　　来自伦敦的青年乐队The xx，2009年首张专辑《xx》以初生之犊的姿态，拿下欧洲指标性水星音乐奖，随后几年间，他们纵横各大音乐祭与颁奖典礼，第二张专辑《Coexist》挤下The Script、Bob Dylan、Pet Shop Boys空降英国流行专辑榜冠军，此等天之骄子般的惊人声势，以及身处譁众取宠的当代音乐潮流中，The xx徘徊午夜的极间低迴旋律铺陈起的黑灰色阶，似乎显得格格不入，然而吸纳著Siouxsie and the Banshees、The Cure、New Order、Cocteau Twins以至Aaliyah音乐养分而来的浪漫声响，却敲醒了层层冰雪底层的炙热，写尽了这时代人们藏得最深的孤寂。

　　2014年乐队灵魂人物Jamie xx首张创作专辑《In Colour》，在The xx失去时空标记的未来场域，投入大量多元声响取洋的缤纷色泽，让人预见了The xx的可能性，亦塑造出暌违五年新专辑《I See You》首支单曲〈On Hold〉的蜕变，与Hall & Oates 1981年冠军曲〈I Can’t Go For That〉交会之际一脚踩进舞池，让好似被时间消磨了的遗憾，不再那麽难以承受，冬阳洒落的和煦蔓延至〈I Dare You〉与〈Say Something Loving〉，依旧浪漫得执拗。

　　开场曲〈Dangerous〉强悍bassline与管乐驱策下，扬起了The xx至今最果抉、激昂的一回；〈Lips〉取洋自葛莱美提名挪威人声乐队Trio Mediæval的脱俗吟唱乘著Tribal Beat悠缓跃动的宜人风景，以及〈Performance〉凄薄弦乐于一片荒芜中拉扯出的张力，格外令人惊豔；即便〈A Violent Noise〉和〈Brave For You〉回到那个被喧嚣遗忘的街角，遇到我们熟悉的The xx，冰冷得暴戾的梦，也相对辽阔轻盈，美得令人屏息。

　　末曲〈Test Me〉绝望的最后恳求，在一片极间冰凝的苍白电气中持续坠落，一段Alessi Brothers〈Do You Feel It?〉的琴音乍现，对映著取洋了同一首歌的〈Say Something Loving〉，麻木了心，模糊了时空，即使在阳光洒落的地方，The xx依旧以近乎任性的执著，耽溺于怀疑与悔恨的结，最亲密的疏离，最炙热的冷冽，让人倚著冰牆试图取暖，距离心一步的位置，目光犹疑试探，却怎麽也踩不进去的怅然。
