简介

　　Sound Horizon被成为「幻想楽団」。与一般的音乐不同，他们创作的主要是歌唱讲述幻想故事的组曲般的「物语音楽」。肯定有不少人熟悉这个乐团，因为他们带给我们的已经远远不只听觉上的震撼。而这张 Elysion～楽园幻想物语组曲～ 是我最喜欢的专辑。不管是熟悉还是第一听说他们的人，让我们重温这份感动！

　　我们可以说全部故事的主人公就是大叔Abyss(也就是Jimang大叔~)。故事从「エルの肖像」开始。还是个正太的Abyss叔叔某天看见了Alice的肖像。被纯美的Alice吸引的Abyss，戴上了假面，踏上了寻找Alice的旅途。

　　在寻找Alice的旅途中，Abyss先后遇见了Ark、Baroque、Yield、Sacrifice、Stardust五人。这就是这五首歌的开头都有Abyss大叔的独白“彼女こそ…私のエリスなのだろうか…（“只有她……才是我的Alice吧……”）”的缘故。

　　可是，寻求乐园（Elysion）的五位少女都犯下了罪，反而堕入了深渊（Abyss）。在她们堕入深渊的时候，Abyss大叔都会在她们身后出现。但他一定是摇著头失望地离开的吧。

　　紫色的Ark：

　　拥有方舟之名的1096号“妹妹”Ark，怀著对方舟的向往，爱上了被称为“哥哥”的1076号。希望去向无尽的乐园的她，手持匕首嬉笑著刺杀了哥哥：“一起……到乐园去吧……哥哥……”被鞭打，被虐待，被调教之后的Ark又被投进了狭小?暗的牢房。背后，有一双一直监视著一切的双眼……原来Ark和哥哥都只是试验品，如同实验室的小白鼠一般。冰冷的声音报告著一切——“虚妄型方舟依存症候群，即‘Ark’”…… 由於Ark希望和哥哥一起去到乐园，却由於Ark置哥哥于死地。Ark的悲惨爱情，在“监视卿”的眼中，不过只是一个平凡的案例罢了…… 哭泣著的Ark背后，出现了假面的男人的身影……

　　蓝色的Baroque：

　　供奉神圣的主的修女Baroque，在暴雨滂沱的夜深深地忏悔著。“主啊，我杀了人。我将那个对我最重要的女子，亲手杀害了。”从小就有妄想症的Baroque，一直不被人们接受。直到对她温柔体贴的“彼女”的出现。由於“性别倒错性歪曲，即Baroque”的作用，Baroque爱上了这位美丽的女性。但是，在Baroque终于控制不住向她告白了之后，这位女性却断然拒绝了她，并且从她身边跑著逃走。Baroque在她身后急急地追赶，失手将她深爱著的女性推落了楼梯。“这扭曲的心灵，这扭曲的贝壳，我的红色珍珠也已经扭曲了吗？”责问著自己的Baroque，在教堂裏深深地跪倒：“我的告解并不指望得到谁的宽赦。这罪孽的缘由，就是我和她之间的羁绊相牵。这罪孽的深重，即使是神也无法宽赦吧……”终于，Baroque选择了自杀。 再一次的，假面的男人叹息著从?影中浮现了：“那么，就让我来宽赦你吧……”

　　绿色的Yield：

　　和父母亲幸福地生活在一起的Yield，原本是个快乐美丽的孩子。可是她渐渐地与自己的父亲陷入了爱河，并且有了一夜的温存。在被母亲发现之后，Yield痛苦地讯问著：“为什么人们(人)恋爱了　却不能在合适的季节(时候)相遇？啊……爸爸(Dad)……妈妈(Mom)……”残酷的算式，描绘出的是更残酷的现实：“「3」…不安定的数字　「3-1」…标准的算式。问题并不在个体的性质　唯有…作为符号的数量而已……为了使世界安定下来　?快把其中的哪个给排除吧……”哼唱著甜美的歌曲，Yield挥起了镰刀：“harvest harvest it yields fruits…la la, latest harvest it yields sweets…Sweets, lala Sweets, lala 鲜红的Fruits要是摘不到它的话　就砍下来好了……Sweets, lala Sweets, lala 鲜红的Fruits啊……可那不是头颅吗……”“两个♀(女人)　一个♂(男人)　其中最不幸的是谁？落下的果实…滚动的声音　余剩的数字…排除的声响……” “「3-1+1-2」” 收割了双亲头颅的少女，就这样，得到了永远的“幸福”…… 又一次地，假面的男人出现在了少女的身后……

　　黄色的Sacrifice：

　　原本嫉妒著温柔善良又受欢迎的妹妹而对其暗自诅咒的Sacrifice，在一场疾病夺走了妹妹的智力之后后悔起自己对妹妹的诅咒来。在那之后，病魔夺走了母亲的生命。两位少女就这样相依为命。在姐姐努力工作的同时，受著村裏女人们的照顾与白眼，失去正常智力的妹妹却被村裏的男人们恣意玩弄。终于，妹妹怀孕了。女人们把嫉妒全部发泄在了她的身上，男人们也只是冷眼旁观。终于，妹妹被宣判为怀有与恶魔作交易得来的灾祸之子种，被施以火刑。在火光中，妹妹却只是微笑著，对姐姐Sacrifice说了一句“谢谢”…… Sacrifice终于暴走了：“啊……真正的恶魔，不正是你们这些家伙吗！”妹妹微笑著?恕了的一切，自己却绝不能?恕啊。“这个世界毕竟，只是乐园的代用品不是吗。罪孽深重的家伙们全都，一起归于灰烬就好了！”终于，Sacrifice点起了火把…… 在那火光之中，现出了假面的男人的身影……

　　红色的Stardust：

　　赤红的唇膏，赤红的洋鞋，赤红的衣裙，赤红的蔷薇。美丽的Stardust沉浸在与恋人一起的幸福中，哼唱著轻快美妙的歌曲。恋人炽热的情话还在耳边：“美丽的星尘……比不上你的美丽……”可是机缘巧合，Stardust碰巧看见了自己深爱的恋人与另一个从未见过的女人身著白衣相拥而行的画面。Stardust的“裏人格”觉醒了……她用手枪射杀了自己的爱人和第三者。“你那白色的衣服(Shirt)　现在已是鲜艳的深红(Scarlet) 在一起的我们　这样就在一起了呢　啊，幸福……” 可是，从裏人格中清醒过来的Stardust，在看见自己亲手酿成的一切之后，无法抑制自己的哀痛，爆发了撕心裂肺的悲鸣：“为什……为什么……为什么啊——！！！！”一声枪响，名为星尘的女人亲手让自己成为了闪耀的Stardust。 假面的男人不知何时已经握住了女人冰冷的双手……

　　作为对ABYSS五位少女的总结，「エルの絵本[笛吹き男とパレード]」是以总是出现在堕入深渊后的少女们背后的Abyss大叔的口吻，以绘本为载体，唱出的“魔性之音”。

　　“譬如那信从着方舟的少女……譬如那扭曲珍珠般的女郎……譬如那错误地收获的女儿……譬如那妹妹被牺牲的姐姐……譬如那为星尘操纵的女人……

　　没有人能从假面的男人ABYSS那里逃脱……

　　‘日安，可怜的小姐啊。欢迎来到乐园的游行！

　　被笛子的声音诱惑　一个又一个人加入队伍之中很快这游行的行列　就把背叛夕阳的地平线全部烧灼……”

　　歌词中的五个“譬如…”就是对ABYSS五位少女的具体写照。

　　而这首歌也表明了加入“乐园的游行”从而堕入“深渊”的人们还在不断?加。而且他们永远不可以离开这罪恶的深渊。

　　PS：Ark Baroque Yield Sacrifice Stardust首个字母连接起来就是 “ABYSS”。

　　嗚呼…果たして其の世界に君が生まれるに足りる浪漫はあるのだろうか?

　　【アーティストについて】

　　領主こと、サウンドクリエーターのRevoが主宰するアーティスト集団であり、自ら[幻想楽団]と称してはいるが、既存の《楽団》とゆう考えとは意を異にする部分もある。

　　現在、会員数が1,600万人を超えるニコニコ動画内では人気楽曲の「歌ってみた」動画が数多く投稿され、非常に大きな盛り上がりを見せております。

　　動画内でパフォーマンスをしている人達は総称して“歌い手”と呼ばれ、人気歌い手の一人である“バルシェ”のALBUM(3/24発売)はオリコンデイリーチャートで7位に入り「歌ってみたアーティスト」の人気の高さを証明しました。そんな状況下で同じく人気歌い手のclearが遂にソロALBUMをリリースします。ニコニコ動画に投稿され反響を呼んでいる楽曲、clear本人が作詞作曲した楽曲などを全曲歌い直し、ニコニコ動画に既に投稿されている音源とは差別化を図ったALBUMです。
