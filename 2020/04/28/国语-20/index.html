简介

　　Kelly数码专辑《Watch Me》合集广东和国语歌

　　多变演绎！「气质女声」被受高度评价

　　破天荒八首歌八个MV 将听觉与视觉享受推上高峰

　　陈慧琳(Kelly) 继2015年推出实体大碟《And Then》后，今年推出数码专辑《Watch Me》，合集四首广东歌和四首国语歌，Kelly希望用歌声锁着乐迷的灵魂，将焦点放在她身上，呼唤各位认真地「Watch Me」！

　　气质女声 像珍珠绽放异彩

　　《Watch Me》专辑里共有八首全新作品，当中四首是广东歌，包括第一主打《尾站天国》、第二主打《Be With You》、第三主打《花见小路》，和《冷淡》。专辑由一首令Kelly念念不忘的旋律《尾站天国》打响头炮，讲述一段恋情未能到达尾站开花结果，因为爱人的离开，只剩下Kelly以缱绻心酸的声音唱出怀念对方之情。之后推出的《Be With You》，当思念去到最严重的地步，就是幻觉到你思念的人在你的身边，Kelly温纯地说唱出与情人分开后的寂寞之感。到后来的《花见小路》，Kelly回忆一段逝去的感情，无奈地唱出对昔日片段的怀缅之情。谁知一开口就被韩国作曲人惊叹Kelly的声音「好像珍珠脱蚌而出」，Kelly当场听到极高评价后，感到愕然不知如何反应。接连几首派台新作，都可见出Kelly以不同的情感和技巧演绎，发挥得淋漓尽致，难怪这把「气质女声」受到韩国作曲人的高度评价！专辑里另外四首国语歌，分别是《打字机》、《答应你会幸福的》、《做自己的太阳》和《也许我一直不懂爱情》。其中《打字机》是一大惊喜之作，唱出「打工一族」的心声，上班下班判若两人，Kelly分别唱出工作时的诉苦埋怨和下班后的狂欢尽兴，完全代入其中。

　　破天荒八首歌八个MV

　　不论是主打歌与否，《Watch Me》专辑内每一首作品都有拍摄MV，这是十分奢侈的做法，同时见出每一首歌都是Kelly和制作团队们的心血结晶，希望将这份心思化为影像留在乐迷的心中。Kelly的疗愈歌声，带有一股特别的魔力震摄人心，把每一首歌曲都配合MV影像诠释出来，听觉配合视觉上的享受，更上一层楼！其中，《尾站天国》和《花见小路》MV取景日本，一个Kelly十分熟悉而且留下不少脚印的地方，画面唯美。值得一提的是，Kelly在《打字机》MV里饰演主角「打工女皇」，又唱又跳之余还一边演出「打工仔」的心声，维妙维肖，绝对是令人眼前一亮，忍不住「Watch Kelly」!

　　2016年薛之谦《初学者》专辑隐情推荐第二抒情主打《我好像在哪见过你》

　　继在市场上叫好叫座的《初学者》与《刚刚好》的曝光以后，薛之谦的全才创作歌手形象再度刷新大家对他的印象。尤其《刚刚好》这首标准的“薛情歌”让广大乐迷们再度跌入薛之谦的抒情魅力中，纷纷在敲碗询问还有没有下一首情歌可以一解情歌之渴。这首再次由薛之谦词曲创作的《我好像在哪见过你》即将让大家重新认识情歌的薛之谦。《我好像在哪见过你》有别于《刚刚好》的深情婉约，所带来的情感浓度是更加浓稠的，歌词的遣词用句也更加趋于意识形态的展现。这首歌的听觉氛围就像一个人在深夜眺望一片深蓝色的海洋，视野尽头表面上看似风平浪静，但实则海平面下却暗潮汹涌，蕴藏着情感的漩涡。歌曲的音符缓缓揭开一段尘封的爱情故事的序幕，让人看见更趋广角的成熟男人的内心风景。这首歌的歌词想说的是“在爱的人的面前压抑住内心漫溢的情感”的柏拉图式爱情。碍于现实的限制，两个有过故事的人，在往后的人生再度相会了，可是却无法再续前缘，只好把澎湃的爱深埋在心里，把对方当陌生人一样看待。就像歌词里提起的“人们把难言的爱都埋入土壤里 袖手旁观着别人尽力撇清自己”、“我听见了你的声音 也藏着颗不敢见的心”、“你听不到我的声音 怕脱口而出是你姓名”一样，这种复杂却让人浮想连翩的爱情故事，是经历过时间洗礼的成熟男人才会体悟到的悲哀。薛之谦的这首《我好像在哪见过你》会让你听见更具深度与想象力的薛情歌！

　　薛之谦这首《我好像在哪见过你》同时也是动画电影《精灵王座》的主题曲，歌曲细腻的情感也很好的诠释了影片中人类少年小鱼与精灵女友莉雅这对“跨族CP”重逢后无惧界限，生死相依的强烈情感。而这部由《龙之谷：破晓奇兵》原班人马打造的魔幻爱情动画电影《精灵王座》也即将在2016年8月12日上映。

　　8/2 华丽转身 空降真我

　　这一次要为爱 交出完美的答卷

　　华纳音乐力推「绝美新声」Loura娄艺潇

　　强势加盟华纳音乐 推出个人首张同名专辑「LOURA」霸气夯实「歌手」身份

　　娄艺潇从来都是令人惊艳的那一个，她是《爱情公寓》里机灵古怪、广受观众喜爱的胡一菲，她是当下热门综艺《跨界歌王》里唱功一流的“灵音唱将”，也是近期大型音乐剧《周日恋曲》里一次连唱24首歌曲的当家主演。她会钢琴、吉他、会跳舞、做得了词也合得了音，不同的人可能因不同的方式认识她，但都会喜欢上这个才华与美貌并存的全能实力派。

　　综艺节目让娄艺潇初次显露了她的出色唱功，但其实本是音乐剧专业科班出身的她一直拥有出类拔萃的演唱实力，因此她能在近期出演大型音乐剧《周日恋曲》，全程没有一句台词、只能通过演唱和表演来传递复杂的情感，也就不令人感到意外了。不过上综艺、演音乐剧，终究并不能完整展现娄艺潇内心丰富的音乐世界，因此她选择跳出荧幕女神形象，霸气宣布「歌手」身份，更像是绕了一大圈之后回归了歌者的老本行。

　　继去年高调加盟华纳音乐并推出首支单曲《野生情人》大获好评以来，娄艺潇的首张专辑一直都在紧锣密鼓的筹备中。这一年，她投入了大量的精力在音乐事业上，尽可能做到每一个细节都亲身参与，亲力亲为，甚至连全国巡演音乐剧也都成了她不断修炼音乐素养的机会，舞台上的学院派唱法、高标准的技巧要求、全国多场实战巡回演出，无形之中也影响拔高了娄艺潇对流行音乐的把控能力和演绎水准。

　　「专辑终极命题——什么才是真的？

　　所有的出发都可能回到一个原点，

　　感受到的，才是真的，是真的，才是最美的。」

　　无论是参与创作的专辑曲目，还是MV脚本策划，娄艺潇都力求融入她的感受，以及在艺术造炼下展现出来的最真实的她——那个定义了“多才多艺”的她。

　　「Scott Effman x Tushar Apte x Stuart Crichton x Steve Shebby

　　黄韵仁 x 杨杰 x 金大洲 x陈韦伶 x 夏侯哲x 黄鹏程 x 管启源

　　—— 欧美知名制作人、华语资深创作人、乐坛新贵音乐人 一齐贡献绝佳创作」

　　落实「歌手」身份之后的娄艺潇，自然要在音乐上施展自己的雄心壮志，希望自己首张专辑可以是一份令人满意的「答卷」，充满丰富精彩多元化的曲风，全面展现娄艺潇的音乐可塑性。专辑除了一口气邀请了Scott Effman、Tushar Apte、Stuart Crichton、Steve Shebby等欧美乐坛炙手歌曲制作人前来打造前卫时髦的作品，更有新加坡金牌制作人黄韵仁、内地音乐人杨杰、新锐音乐人金大洲、内地鬼才音乐人黄鹏程前来加盟负责专辑词曲、制作的打磨，台湾“音乐女诗人”陈韦伶、内地资深音乐人夏侯哲、资深词人管启源也贡献了自己珍藏已久的佳作……网罗欧美、华语各路知名音乐人，只为打造令人期待已久的「绝美新声」。

　　「十首歌 六支MV 一本写真 华纳音乐 重金打造

　　美国洛杉矶、索尔顿湖、台湾、上海 —— 远赴多地拍摄

　　黄中平 x 黄子然 x 谢韵 x 马锐 ——联合倾力打造」

　　为了呈现最美的娄艺潇，华纳音乐也不遗余力重金打造，10首单曲完成6支MV，远赴美国洛杉矶、索尔顿湖、台湾、上海等地拍摄，同时推出一本全新个人写真；邀请著名导演黄中平、黄子然、新锐导演谢韵、亚洲顶级美学专家马锐等资深大咖鼎力合作。

　　首波主打「非类」的MV力邀台湾资深鬼才导演黄中平掌镜、台湾偶像剧标配男主陈楚河参演，娄艺潇和工作人员更远赴台湾宜兰进行了拍摄工作。MV采用意识形态的表现方式，展现出在爱情逐渐消失的过程。拍摄日恰逢雨季，阴雨连绵，也打乱了一些拍摄计划，导演不得不在现场修改了部分拍摄内容，潇潇也主动提出了自己的想法，更给导演的脚本锦上添花。

　　作为专辑的第二波主打单曲，「好想他」是一首朴实深情、格外走心的情歌。歌曲MV由台湾新晋导演黄子然拍摄，内地男演员李子峰加盟，与娄艺潇一起上演了一个温存美好又令人心碎的爱情故事。娄艺潇坦言：“这首歌和故事，十分令人感同身受，看到自己曾经的影子。”

　　【歌曲介绍】

　　「非类」

　　作为专辑的第一波主打，「非类」是内地词人杨杰和美国音乐人Stuart Crichton、 Sidnie Tipton、Meron碰创出的火花之作。曾跨界演绎昆曲的娄艺潇对中国民族戏曲的唱法驾轻就熟，为了能在她的这个优势上加以修饰创新，华纳音乐更重金请来曾为Selena Gomez、Kesha制作过大热金曲的美国当红制作人Stuart Crichton制作这首“既传统民族又结合西方律动”的「非类」，时尚舞曲元素搭配大气节拍，用音符带你一起表达对自我生命的热忱，逃离世俗眼光。存在于这个世界上的每一个人都属于独立的个体，都是区别于他人的「非类」，都有属于自己的故事和精彩，即便外界存在的太多想要改变「本我」的因素，纵使这些“绊脚石”多么的碍眼，纵使在他人眼中是多么的格格不入，我们也要学会坚持，保存属于自己的那一份独特。

　　「好想他」

　　由台湾“音乐女诗人”陈韦伶亲自为娄艺潇作词谱曲，娄艺潇也参与了歌词与创作，更由内地新锐音乐人金大洲制作加持，打造一首充满温度的“潇”式情歌。「好想他」这首歌诠释了感情中分手后，情人离去，却心中烙印深刻的状态。娄艺潇以叙事的口吻将恋爱的心事娓娓道来，契合唯美动人的旋律，一字一句唱出内心最温柔的声音，却也透露着热烈后孤单的丝丝愁绪。爱情的道路难免会有坎坷，美好的回忆与未知的未来都令人难以割舍，时常感恩，心怀念想，感谢生命中曾经的他，那些美好都值得想念，生命也会因此充实而珍贵。

　　「海市蜃楼」

　　作为专辑的开场曲，「海市蜃楼」以恢宏气势与壮丽画面感揭开帷幕。最初收到这首Timers乐队的作品时，公司认为这首歌略显“另类”，但娄艺潇坚持认为这首歌非常特别并宠爱有加，在繁忙工作中都不忘为之亲自填词，甚至在最后一刻推翻全盘制作，只为追求最理想的效果。更邀请了内地鬼才音乐人黄鹏程前来制作打磨，以钢琴作为主线索引，运用大编制弦乐团来呈现作品的厚重与壮丽，电子音效和实验人声的交织，犹如穿梭于不同时空维度，体验浓厚的历史探索感。歌词中出现的不少古代神话意象，在娄艺潇气若游丝的演绎中如同海市蜃楼般被徐徐构建起来，象征着古希腊王朝的建立与战争的腥风血雨。在文明的形成与衰落中，爱情是人类永恒的话题，歌曲也正是从这个时代的角度完成了一次关于历史风雨与爱情意象的追溯。

　　「缱绻」

　　整部《扶摇》，总共给男女主的成长和感情配置了三首主题曲，这样的“音乐匠心”在影视剧里并不多见。除了体现出制片方对细节的追求，对创作者董冬冬和陈曦来说，同样是功力的考验。

　　不同于坚忍不拔的性格主题曲《扶摇》和炽热与险阻抗争的命运主题曲《血如墨》，爱恨交织的情感主题曲《一爱难求》听感上更加通俗和流行。创作和编曲上所强调的细节精致，让音乐显得非常细腻和丰满。《一爱难求》肉埙、琵琶等民族乐器缓缓交织古韵悠然，点缀的流行乐器完美配合激起听觉的阵阵涟漪。而徐佳莹的声线，则是其中绚烂的花火。收的时候低回而克制，犹如听者内心的自问：“你用什么/换这一爱难求/若无病弱/若永无苍老/又怎叫花火/孤身逐星斗”；而放的时候，那种大情感的表达更注重含蓄内敛，不是直接冲击式地打到你，环绕声般一浪一浪地包着你，卷着你到一场传奇中。

　　在制作人董冬冬看来，《一爱难求》这首歌表达的是扶摇这个人物的爱情观，她的爱情是一种独立又坚韧的爱，和无极并肩而立，相互鼓励一起成长。一方面她深知爱情的可贵，“爱是不能分享的，喜欢一个人就要一心一意”，另一方面却并不会为爱情停下自己前行的脚步，听起来 “一爱难求”，但实则是“一生一世一双人”。在“矛盾”中行进，却又在共鸣上达到高度的统一，正是《一爱难求》最大的特色。比如民族乐器与现代乐器的碰撞交织，以及徐佳莹声线中低沉婉转的情绪间的自如切换，无不突显着这一点。

　　而这种爱情观和价值观的背后，也体现着当下年轻人敢爱敢恨勇敢逐梦的赤子之心。在刚刚结束的中华音乐人交流协会2017年“十大专辑暨单曲”评选中，徐佳莹凭借专辑《心理学》获奖，评审团评价她为“兼具艺术与商业的成熟音乐人”。在《一爱难求》的录制过程中，徐佳莹同样体现了这种成熟度。Demo给过去不超过五天，董冬冬就得到了徐佳莹肯定的答复，过程中并没有再做更多的沟通，全部录音部分都在台湾完成，最后发过来的成片却令人惊艳。在歌曲的后半部分，徐佳莹甚至还自己设计了一段和声，同样起到了点睛的效果。

　　《扶摇》由柠萌影业，柠萌悦心，企鹅影视联合出品，改编自天下归元小说《扶摇皇后》，解嬿嬿编剧，杨文军、谢泽、李才联合执导，杨晓培担任总制片人，杨幂、阮经天领衔主演，刘奕君、王劲松特别主演，高伟光、赖艺、张雅钦、胡可、高瀚宇、黄宥明、李依晓、蒋龙等联袂主演，腾讯视频会员每周一至周四抢先看四集，浙江卫视中国蓝周播剧场每周一至周四22：00两集连播。

简介

　　《唱游》是第一张“HDCD”华语专辑，所以，也是华语流行音乐的新纪录！对于王菲的音乐，大家一直充满了期待，这一次王菲要给我们什幺？但是，我们深深知道，曲风、音色、唱腔……等的改变、尝试或创新，都还是后话；因为对于整个音乐本质的改进，才是根本提升音乐的第一步。所以，王菲的《唱游》，就是竖立新音乐标准的新尝试，也不负一整年的期待。“HDCD”是这张专辑的第一个大礼。

　　《唱游》是华语流行音乐市场第一张采取“HDCD”的编码录音专辑。目前全世界只有古典唱片或少数讲究真实、立体、层次极饱满的发烧片，才是以“HDCD”方式编译录制完成。而王菲一直是华人音乐的骄傲，也是这张专辑所被赋予的一种使命感。相信也是值得大家鼓励的，在聆听的过程，希望也让大家耳目一新。

　　如果用心听，在这张精致的编曲，非但感到丰富的音乐，还有层次的立体感，更重要的是，在饱满音场中，有前所未有的繁复、绝不混淆。过去的“迷幻”类型音乐，一直是讲究氛围，各种音色〈如吉他、贝斯，甚至人声……等〉在效果器扩大效果后，让音乐的层次、音场含糊，如果以过去的录音编译方法，这些层次、立体感、饱满的音场及透明度与气氛，常常让人有种鱼与熊掌，捶胸顿足之憾。“HDCD”足以将这张的音色、人声，展现出透明度，正是一重要的突破。
