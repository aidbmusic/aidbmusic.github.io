简介

　　周杰伦的出现，让人们相信台湾创造本土R&B的可能性；周杰伦的走红，彻底地宣布音乐新声代的来临。作曲、填词、编曲、演唱样样俱精的周杰伦，首张同名专辑《Jay杰伦》推出后，销售势如破竹，不单有“音乐新人王”称号，他自成一格的R&B演绎方法，更被誉为陶吉吉的劲敌。 顶着台湾金曲奖“最佳唱片”、“最佳制作人”的光环，周杰伦的第二张专辑《Fantasy(范特西)》终于在万千歌迷的期待中推出。这次杰伦依然一手包办所有作品的作曲，词的部份则延续第一张专辑的徐若？、方文山及他本人的铁三角。曲风较上张专辑更为的广泛，除了抒情歌及R&B外，杰伦还首次在歌中加入他在音乐班主修的大提琴，更有类似林普巴兹提特的摇滚及Hip-Hop曲风。 如今身价过亿新台币的杰伦在与BMG续约两张专辑的发行权后，有了更多的创作空间，在《范特西》中天马行空的音乐想法无处不在，就连歌名也都是令人意想不到的饶富趣味。有专门讲述日本忍者的“对不起”、因为喜爱看中国的武打电影而创作的“双截棍”等。第一主打曲“开不了口”由徐若？填的词、杰伦作曲，是一首描述生离死别的抒情作品。在Music Video中杰伦扮演一名宇航员，但却不幸意外身亡，之后又经历犹如电影《第六感生死恋》的天人永隔的无奈，故事情节令人心碎。 成功的跨出歌坛的第一步后，周杰伦成为现今台湾流行乐坛最抢手的创作人，而紧接他将会在11月举行大型的售票演唱会，年底还将大举进军日本，发行日本专辑，由此可见周杰伦的实力及威力都非常的惊人。

简介

　　一直以音乐制作人身份从事作词、作曲、编曲等工作的周杰伦，18岁时因创作吴宗宪的“屋顶”、“三暝三日”、“你比从前快乐”；许茹芸的“蜗牛”；王力宏的“打开爱”以及徐若宣的“姐你睡了吗?”等歌曲打开了知名度。害羞、内向似邻家大男孩的他凭着新作“可爱女人”的强力宣传，从幕后走到幕前成为歌坛耀眼的新人，首张同名专辑《周杰伦》于2000年11月3日在台湾正式发行。

　　专辑中周杰伦一手包办了10首歌的作曲、合声编写、3首歌的编曲以及2首歌的作词。整体曲风充斥着他对音乐的超高技术创作，以R&B和New Hip-Hop的新曲为主，加上古典巴洛克式弦乐伴奏及Band的加入，形成一种英国式的复古风格。最特别的是尝试了西班牙式风格的弦乐演奏，表现在专辑歌曲中，意境却出忽意料的极度逼近电影配乐。 主打歌“可爱女人”的Music Video邝盛执导，徐若?出任女主角，导演巧妙地安排给两人一段有想象空间的剧情桥段。

　　由于周杰伦与徐若宣是多年好友，又加上在音乐上多次的愉快合作，徐若宣特地为这张专辑的三首歌填词。其中一首“伊斯坦堡”更是她在亲临土耳其伊斯坦堡时所写。两人密切的合作，相信在听觉上、视觉上必能带给歌迷前所未有的新鲜感受。 在大众都已进化到音乐所带来的听觉感官十分细腻的同时，便需要周杰伦这样新的创作。不再只是概念式叙事的音效，及民歌情调流行乐和新电子风，听他的音乐，可以是幻想的，幻想着快乐，超越空间与时间的束缚，是惊讶的、是舒服的。

简介

　　欣赏JAY复古新造型：

　　更有深度�周杰伦《叶惠美》

　　近日转会加盟新公司的周杰伦，同时推出了其年度国语专辑《叶惠美》，收录11首个人独唱歌曲。虽然新碟的惊喜不大，但仍保持着阿Jay一贯的个人风格(其实这正是乐迷最喜欢的)，而且多首歌曲亦较以往更有深度，故此并不算「原地踏步」。再者，当中某几首歌曲，更是惹人钟爱之作，相信这专辑依然会得到不少乐迷支持。

　　「以父之名」故事性强，歌词用了很多不同的景物去呈现歌中「教父」那懊恼与痛苦的心境，令听者能有更立体更实在的感受。此歌的编曲方面亦富歌剧味道，与歌曲的整体感觉与主题配合。

　　近期经常在电视上听到的广告歌「你听得到」，是首较大路的作品，旋律流畅动听，歌词则温馨但不特别。曲词动人的「爱的悬崖」运用了大量的弦乐，阿Jay的诠释富感情，能表达出歌中的无奈与伤感。作词人徐若�近年填的词都讨好，言简意远，可惜题材不广，未能在词坛创作更大成就。

　　「三年二班」写出许多朋友在年青时的疑问与心声，整首歌让人有一种说不出的感受，勾起了听者许多许多的回忆。仿古的「东风破」不仅曲子有味道，方文山写的歌词更精彩，非但用字优美，一向注重韵脚的他，填得很有韵味。

　　揉合了中乐、少数民族音乐及西方乐器的「梯田」，唱出「物换星移，故地不再」的唏嘘，不过头几句揶揄方文山写词慢的几句，真的十分搞笑。「双刀」是阿Jay专辑必备之作，正如前作「双截棍」及「龙拳」般，总觉得阿Jay中李小龙毒太深，此歌的吸引力只属一般。

　　其它歌曲方面，情歌「晴天」、写一见如故的「同一种调调」及摇滚味重的「懦夫」亦不容错过。

　　谁是「叶惠美」�

　　周杰伦 (Jay) 每次出碟，除了在音乐上带给大家惊奇外，就连专辑名称都会让人有无限的想象空间，如之前《范特西》、《八度空间》。这次Jay的新专辑，光看专辑名称「叶惠美」就够吸引了，但也让歌迷在心中打了一个大问号，究竟「叶惠美」是何许人也？可以让Jay用他的名字作为专辑名称？原来在他的新专辑中第一波强打主打歌为「以父之名」，为了以示公平，想象力丰富的Jay灵机一动，就用母亲的名字「叶惠美」作为专辑名称来平衡一下，因此专辑名称「叶惠美」原由在此。

　　周杰倫（ジェイ・チョウ）が2003年に発表した4thアルバム「葉惠美」が、ミュージックビデオやドキュメンタリーを収録したDVD付きで再リリース。　この「葉惠美」というタイトル、実はジェイのお母さんの名前なのだそう。というのも、第1弾フィーチャートラックが「以父之名（父の名において）」に決定したことから、お父さんが出てくるならお母さんにも登場してもらわないと不公平、というジェイ独特のアイディアでこのタイトルに決定したのだそう。アルバム全体の雰囲気は、ジャケットにも現れているとおり゛レトロ゛なムード。オンエアトラックの「以父之名」（父の名において）は、カンツォーネやオペラなど、イタリアンな要素を取り入れた楽曲。ミュージックビデオでは、ローマ・ロケが敢行された。ゴッドファーザーを彷彿とさせるこのビデオで、ジェイは父の仇を狙う殺し屋を演じている。

简介

　　“亚洲天王”周杰伦最新专辑“十一月的萧邦”，周杰伦自己打理造型化做音乐诗人全新亮相，宣告华语音乐的浪漫古典时期来到，由于这次亮相的狂野复古华丽风的造型，并与专辑主打的浪漫古典诗人形象不符，但周董得意的说：很喜欢这样看似狂野的造型打扮，虽然不会出现在专辑内，但因为自己很满意，所以将以限量方式作为预购的赠品海报。

　　原本每年暑假发片的亚洲天王周杰伦，今年夏天并没有如同以往的发片，周董解释，这不是缺席，而是希望音乐在制作上能够更加精细。今年迟了一点，但，感觉却也多了一点！周董的第六张国语专辑“十一月的萧邦”，选择在具有浓浓秋意的十一月一号发行，虽然比往年迟了一点，但是大家可以在专辑当中，嗅出杰伦潜藏的浪漫古典因子，看到他多放了一点细腻的感觉在音乐里，耐听度更高，“十一月的萧邦”将会让歌迷发掘不同面向的周杰伦，为营造秋天的气息并符合“十一月的萧邦”专辑，富有浪漫时期味道的感觉，杰伦特别选择俯拾即是艺术的“水都”威尼斯拍摄专辑照，在没落贵族的华丽城堡当中取景，陈列在其中的精致画像栩栩如生，周董还开玩笑的说：“好像置身在“哈利波特”的奇幻世界当中。”而已经自己做了好几张专辑造型的杰伦，这次还冒着行李超重的危险，特别准备了高达二十几套的服装，期望在专辑照当中完美扮演萧邦的角色，此举也不得不让人佩服亚洲天王的用心良苦！

　　周杰伦的音乐一向给人强烈的画面感，由于新专辑中的每一首歌都充满了诗意，营造出诗境般的画面，让人联想到“钢琴诗人”萧邦，一直是学习古典音乐出身的杰伦，非常欣赏的音乐家，而杰伦这张新专辑，恰巧每首歌的歌名均极富诗意，如诗般具有画面的音乐作品，散发出浓厚的文学气息，这跟一向习惯以“诗人”为名来作造型的杰伦，可说是再贴切也不过了，所以当唱片公司提出以“十一月的萧邦”，作为十一月发行的新专辑名称时，双方可说是一拍即合！如果要说当今华语歌坛的“音乐诗人”，杰伦真的可说是当之无愧！

　　从“叶惠美”专辑开始，周董就开始打理自己的专辑造型，这一张专辑“11月的萧邦”造型也是自己一手包办，杰伦说：“这次造型大致是从古典出发。”而周董购买衣服时品牌和价钱不是重点，看到喜欢的他就会买下，尽管当下用不着也会收藏起来以备不时之需，这次在威尼斯取景的时候周董就拿出许多自己的压箱宝，包括照片中这套金色西装外套就是今年春天在香港买的！在威尼斯待了9天，周董说：“威尼斯的每一个地方都很美！”加上遇到秋高气爽的好天气，天气好周董拍照心情也很好，拍下很多很棒的照片！

　　⊙破天荒12首作品超值专辑 加赠三支 MV 与浪漫桌历

　　“叶惠美”专辑是周杰伦对妈妈表达孝心的最好礼物，而“十一月的萧邦”则可说是周董对于偶像萧邦，献上的最崇高敬意！周杰伦即将在11月1日推出最新专辑“十一月的萧邦”，今年虽然迟了一点，但歌迷却可以感受到杰伦在音乐制作上更精益求精的细腻！甚至远赴威尼斯拍摄的封面都可以嗅到杰伦潜藏的浪漫古典因子！为了呈现最棒的封面与专辑照片，唱片公司可以说是不计成本，不惜耗资100多万进行拍摄，这也创下了周杰伦拍摄专辑照片的最贵纪录！

　　这款浪漫古典的封面拍摄地点，位于意大利威尼斯的“圣马可广场”，为了避开拥挤的观光人潮，周杰伦和工作人员特地起个大早，凌晨6点钟就在无人的广场上拍照，一行人抵达时连广场上的鸽子都还没醒来，在开拍后没多久，鸽子就像是活道具般的不停来回飞翔，成了周董新专辑封面的最佳帮手，特别的是为了要一台古董咖啡色钢琴入镜，工作人员是煞费苦心，在威尼斯当地遍寻不着理想钢琴之下，不顾一切的让封面上这台钢琴坐了5个小时的船从米兰运来！拍摄前一晚摆置广场时还得请人顾守呢。而这次杰伦在专辑造型上走的是“华丽宫廷诗人”的路线，视觉上的奢华与俯拾皆是艺术的“水都”威尼斯相互辉映！

　　拍照费用之惊人，不含机票住宿就花了100多万元，创下周董拍照的最贵纪录！主要是因为威尼斯很多地方取景都要租金，包括封面的圣马可广场、歌剧院、华丽城堡、贵族大厅……，不过城堡里一间主人从不开放给人拍摄的房间却因为周董弹了一手好琴而破例！当时工作人员发现城堡中有一个主卧房非常漂亮，不过这间主卧房却是有钱也不能借来拍，正当工作人员忙与主人交涉的同时，传来一阵悠扬悦耳的钢琴演奏，主人好奇的找到弹奏钢琴的周董，直夸他钢琴弹得非常美妙！工作人员此时赶紧介绍杰伦正是要入镜的主角，很懂得欣赏艺术的主人是二话不说，马上破例让周董一行人进入拍摄免费取景！

　　周杰伦日前拍摄电影“头文字D”而入围金马奖最佳新人与原创电影音乐，为这部电影量身打造的“飘移”与“一路向北”两首歌曲也被歌迷热烈讨论，唱片公司决定在周杰伦推出的最新专辑里，破天荒的以 Bonus Track 的方式加收于专辑当中，成为一张十二首作品的超值专辑，也是杰伦发行六张专辑以来，收录歌曲最丰富的一张！同时随专辑将附赠三首音乐录影带 VCD，包括“夜曲”、“一路向北”以及在28日即将于 Channel【V】和 Wave Radio 首播的“发如雪”三支 MV，而在“水都”威尼斯拍摄的许多珍贵照片将制成“周杰伦2006年浪漫诗篇桌历”，在视觉加分之外，满足死忠歌迷拥有周杰伦的欲望！

　　⊙古典吉他与嘻哈饶舌创意融合 21日全亚洲同步首播“夜曲”MV

　　新专辑除了到威尼斯取景之外，首波主打“夜曲”更是远赴纽约拍摄 MV，整张专辑光是视觉上的丰富就横跨了欧美！“夜曲”MV 拍摄的时间是在2004年12月，周董迫不及待飞往纽约拍摄 MV，捕捉寒冬萧瑟的凄美景象。12月结束 L.A 与康州的“无与伦比”巡回演唱会后，一行人即从康州前往纽约郊区，前一晚的大风雪让工作人员差点却步，深怕车子开到一半就被深厚的积雪困住，不过在周董的坚持下大家还是咬牙前往拍摄地点，所幸出发的凌晨大雪渐歇，尽管花了近5个小时才到目的地，总算是可以顺利开拍。拍摄时零下6度的天气还让桌上不小心倒出来的热水瞬间变成了冰柱！严寒的程度可见一般！

　　“夜曲”MV 在纽约市区与郊区的墓园取景，棚内部份则是日前在台湾完成，为了连戏当时的服装也从衣柜中翻了出来，片厂也搭起类似纽约家居的景，拍摄时周董回想将近一年前在纽约拍摄 MV 的情况，不禁开玩笑说：“过了一年，我家还是没变！我还穿着一年前的西装！”周杰伦在看过完整的 MV 成品后表示：“这首歌除了听会感动，看了 MV 也会感动，很屌！看了就知道！”

　　“夜曲”MV 描述男主角为了怀念逝去的恋人，选择弹奏当时女主角最爱的萧邦夜曲，纪念已经死去的爱情。从歌词故事到 MV 都是出自周董的构想，这首 MV 也是周杰伦所有音乐作品当中，首度描写女主角已过世的歌曲！因此不论是歌曲或是 MV 都极具戏剧张力！这首“夜曲”的歌名就是源自萧邦的知名作品“夜曲”，是专辑最早完成的作品，也是整张专辑精神的所在！歌词中“为你弹奏萧邦的夜曲，纪念我死去的爱情，跟夜风一样的声音，心碎的很好听…”方文山写的歌词精准的传达周董脑海中的画面，而这首歌的音乐风格融合了古典吉他与嘻哈饶舌，旋律优美又流露古典气息！给人新意盎然的惊喜！周董表示：“吉他的旋律非常有古典味，Rap 的加入让整首曲子很炫，是一首有创意的小调情歌。”
