简介

　　狮子座的Eason,不仅对音乐具有天赋, 连带对语言都有很出色的表现,英文、法文、国语及广东话都难不倒他。同时对小提琴、结他、钢琴等乐器都有所认识；更令人意外就是他在英国大学Kingston University 里主修的竟然是建筑系, 这可以说是知性与感性并齐。

　　

　　 去年国语专辑《反正是我》得到年度十大专辑奖项，获得媒体及市场一致的好评，今年陈奕迅挟着风光，推出2002年国语专辑《Special Thanks to…》，这次陈奕迅十分有自信，制作人马精简，亲自演唱并合声，只用一位制作人（JIM LEE制作并包办录音混音合声编写），只用一位作词人（林夕包办11首歌词），编曲，吉他、贝斯演奏也是由刘志远一个人包办。《Special Thanks To…》企图在音乐走向及型态上统一的风格，有这样金字塔的铁三角组合，《Special Thanks To…》的实力早已让拭目以待。

　　

　　 整张专辑的歌词全由林夕包办，在得到金曲奖后，林夕这次特地闭关为陈奕迅写歌，有恋物情结的故事、小人物奋斗的鼓励、旧情人重逢的感慨、爱情、没有工作烦扰的小幸福，写出一连串的人生故事。据说林夕为了写好词，有两个月的时间不接外界电话，只用传呼机联络，可见两人的交情。陈奕迅说，两人虽不算很熟的朋友，但有种惺惺相惜的感觉。

　　

　　 专辑中所有声音的演唱全部由陈奕迅一人负责，在制作人Jim Lee的指导之下，陈奕迅体会到了将声音发挥到最不可思议境界时的美妙心情，并且透过一次又一次地尝试，挑战不同音域的声度，激发出许多意料之外的声音表情。

　　首批CD加送: 张学友 x 陈奕迅「同舟之情」CD Single

　　特別以uView手机程式观赏MV「主旋律」「任我行」

　　特別T恤版全球限量2013套同步上市，新艺宝出品

　　陈奕迅这次新专辑的话题回归生活，是一张以人与人之间的关系为主题的音乐 作品。新专辑由C.Y.Kong监制，制作班底仍旧是Eason的老搭档，大家可以看到林夕、小克等熟悉的名字。此张新专辑Eason没有参与制作，专辑主题及选歌完全交给C.Y.Kong处理，整张专辑Eason都以第三身的身份演绎歌曲，但在演唱的过程中亦令他有所启发。新专辑收录8首作品，每一首歌都提及不同的人与人的相处之道，每首歌曲都是你身边的发生的故事。

　　专辑第一主打【主旋律】便提及一对朋友相处之道，除了情侣、朋友之间的相处之道。离合、纠缠、争拗、猜忌…是当今人与人之间的相处方式吗？每个时代都有其所属的歌，每段感情都有其所属旋律，人与人的相处就如五线谱上的每个音符，要排列恰当、和谐，所奏出的璇律才婉转动听。

　　【远在咫尺】写出老夫老妻的生活状态。老夫老妻生活令你觉得烦闷，你在怀念你的前度，觉得她(他)才是你的最爱，但若时光倒流，换位思考，你最爱的前度是你今天的枕边人。从歌词得到启发，大家不妨多以欣赏的眼光看待现任，发现她(他)的优点，“一起这种艺术 / 若果只是漫长忍让 / 应感激忠心的伴侣”。

　　励志歌曲【阿猫阿狗】，歌词鼓励时下年轻人不要当自己是“阿猫阿狗”小人物，不要看轻自己，不要随波逐流，不要当自己是小丑，在时代的方舟上，再大的舵盘都要有小螺丝去构筑，再小的人物都有自已发光发热的岗位。专辑中特别值得一提的是歌曲【床头床尾】，Eason谱曲，小克细腻的歌词写出了夫妻的相处之道，不知这是否作者的体会呢！
