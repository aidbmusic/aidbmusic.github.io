Akustisch and Live

信息

表演者: Die Prinzen

专辑类型: Live / Import

介质: Audio CD

发行时间: 2006-02-07

出版者: Bmg Germany

条形码: 0828767584027

豆瓣评分: 8.7

简介

　　这张是他们2006年推出的一张现场专辑，非常值得一听。无论是新歌迷，还是老歌迷，里面的每一首歌都是Die Prinzen最经典、最流行、最动听的代表作！不能错过！我一直觉得欧美乐队的现场是很棒的，果不其然。当然，现场的演绎多少和录音室不同，你可以听到不同感觉的Deutschland，和今年夏天被各个媒体放烂掉了的原版完全不同（我恨这些媒体，因为他们胡乱播放也不介绍，我想除了德国粉和德语学习者，没有更多人了解Die Prinzen吧？但我也爱他们，因为他们选择了Die Prinzen的歌。

It’s Done

信息

表演者: Overground

专辑类型: Import

介质: Audio CD

发行时间: 2003-11-30

出版者: Universal

唱片数: 1

条形码: 0602498659571

豆瓣评分: 7.7

简介

　　来自德国的四人BOYBOND “OVERGROUND”携青春之势轰炸你的听觉。青春之势轰炸你的听觉。虽然这张《It’s done》发行于2003年，但直到最近，网路上开始流传的一首劲暴摇摆的“one for da money”，才让更多中国的听众知道“Overground”这个名字。如果你是blue,mcfly,c21等新生代boyband的fans,你一定也会成为Overground的歌迷。如果你立志于学习德语，Overground也会点燃你学习的动力。Let’s go,让这群德国的男孩洗礼你早已被男孩组合的英文歌麻木的听觉。
